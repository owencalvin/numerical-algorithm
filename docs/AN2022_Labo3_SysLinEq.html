Résolveur de système d'équations linéaires

Cet outil permet de trouver les solutions d'un système d'équations linéaires de la forme AX = B, avec A et B des matrices.
Le fichier JSON représentant une matrice doit être de la forme : 
{
  "dimension": 3,
  "components": [
    [ 1.0, -1.0, 2.0 ],
    [ 3.0, 2.0, 1.0 ],
    [ 2.0, -3.0, -2.0 ]
  ]
}
La fichier JSON doit respecter le format demandé

Contexte

Dans le cadre de notre formation et plus précisément du cours de mathématiques spécifiques II, nous avons réalisé ce projet de programmation. Il constitue le laboratoire numéro 3 du cours.

Le but est de coder un programme permettant de résoudre un système d'équations linéaires. Ce même programme doit pouvoir lire des fichiers JSON qui vont représenter les matrices de notre système. Les solutions de ce système seront ensuite approximées.

Ce travail est mis en compétition avec les autres équipes. En effet, la note sera relative aux performances des implémentations de la classe.

Méthodologie de développement

Comme à notre habitude, nous avons utilisé la méthode du "Pair programming" pour ce projet. Le cours de ce laboratoire a servi de référence et a été complété par les recherches que nous avons fait sur internet.

En ce qui concerne les technologies, nous avons codé le projet en TypeScript, car cela nous a facilité l'utilisation de types et permis d'utiliser une autocomplétion plus avancée durant la phase de programmation.

Différentes matrices ont été utilisées afin de valider nos résultats tout le long du développement. Des exemples avec des résultats connus étaient trouvables sur internet.

De plus, le format du fichier JSON nous a été imposé et est celui précisé en haut de cette page.

Pour ce qui est de l'implémentation, la page Wikipédia sur le sujet a été d'une très grande aide. Cette page se trouve dans les références ci-dessous.

Conclusion et perspectives

Les fonctionnalités demandés ont été implémentées comme demandé et tout semble fonctionner.

Il est sûrement encore possible d'améliorer les performances dans le futur. Pour l'instant nous verrons où se situe notre implémentation par rapport aux autres groupes.
